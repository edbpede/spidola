Spidola is a free, open-source IPTV player built exclusively for Apple TV and Android TV / Google TV.
It is designed from the first pixel for the 10-foot experience, with remote-first navigation,
user-supplied sources, instant search, low memory use, no bundled content, and no in-app monetization.
A shared Rust core handles parsing, storage, search, and networking, while native SwiftUI and Compose
for TV shells preserve each platform's interaction model and playback capabilities.

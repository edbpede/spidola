# M3U fixtures

The golden M3U corpus (TECH_SPEC §10 — the project's crown jewel). Every real-world
playlist bug ever reported is captured here as a fixture with a provenance note. Seeded in
Phase 1 (`core-parse`). Files are scrubbed of any credentials before landing.

Provenance convention: each fixture is accompanied by a short note (what it exercises,
where it came from, what was scrubbed).

# build-libmpv-android

Pinned per-ABI build configuration for **LGPL** libmpv consumed by the Android
`player:engine-mpv` engine (arm64-v8a, armeabi-v7a, x86_64). Checksums pinned; invoked
from Gradle. Implemented in Phase 5.

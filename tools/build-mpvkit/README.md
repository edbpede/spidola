# build-mpvkit

Pinned build configuration for the MPVKit xcframeworks (libmpv + **LGPL** FFmpeg) consumed
by the tvOS `PlayerMPV` engine. The LGPL build flags are committed and verified in CI
(TECH_SPEC §12). Implemented in Phase 5.
